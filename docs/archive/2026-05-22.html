Gated DeltaNet-2: Decoupling Erase and Write in Linear Attention Gated DeltaNet-2：在线性注意力中解耦擦除与写入

🧑‍🔬 Ali Hatamizadeh, Yejin Choi, Jan Kautz

图 Figure · Page 7 of paper

🎯 线性注意力中解耦擦除与写入门控，提升压缩记忆编辑能力
Gated DeltaNet-2 decouples erase and write gates in linear attention for stronger memory editing.
维度 中文 English
❓ 待解决
的问题		 线性注意力模型将上下文压缩为固定大小的循环状态，速度快且内存高效，但精确编辑这种压缩记忆很困难。之前的 delta-rule 模型（如 Gated DeltaNet 和 KDA）用单一标量门同时控制"擦除旧内容"和"写入新内容"两件事，把两个本质不同的操作绑定在一起，限制了记忆更新的表达能力。 Linear attention models compress context into a fixed-size recurrent state, making them fast and memory-efficient — but editing that compressed memory precisely is hard. Prior delta-rule models like Gated DeltaNet and KDA use a single scalar gate to control both how much old content is erased and how much new content is written, which conflates two fundamentally different operations and limits expressive memory updates.
💡 解决
的亮点		 • 创新解耦：引入独立的逐通道擦除门 b_t 和写入门 w_t，取代先前方法中共享的单标量门控。 • 统一泛化：当两个门退化为同一标量时还原为 KDA，当衰减也退化时还原为 Gated DeltaNet，实现对已有方法的统一覆盖。 • 高效训练：推导了带逐通道衰减的分块 WY 算法，并设计了感知门控的反向传播，保持完全并行训练效率。 • Novel decoupling: introduces separate channel-wise erase gate b_t and write gate w_t, replacing the single-scalar tie shared by prior methods. • Unified generalization: Gated DeltaNet-2 subsumes both Gated DeltaNet and KDA as special cases when gates collapse to scalars. • Efficient training: derives a chunkwise WY algorithm with channel-wise decay absorbed into asymmetric erase factors, plus a gate-aware backward pass for fully parallel training.
🧪 实验
内容		 在 FineWeb-Edu 数据集的 1000 亿 token 上训练 13 亿参数规模的模型。基线包括 Mamba-2、Gated DeltaNet、KDA 和 Mamba-3 的多个变体，分别在纯循环和混合架构设置下评估。评估任务涵盖语言建模（困惑度）、常识推理基准，以及使用 RULER 基准的长上下文检索任务（包括大海捞针和多键检索）。 Models were trained at 1.3B parameters on 100B tokens from the FineWeb-Edu dataset. Baselines include Mamba-2, Gated DeltaNet, KDA, and Mamba-3 variants, evaluated in both recurrent-only and hybrid settings. Evaluations cover language modeling (perplexity), commonsense reasoning benchmarks, and long-context retrieval via the RULER benchmark, specifically the needle-in-a-haystack and multi-key retrieval tasks.
📊 实验
结果		 在 13 亿参数、1000 亿 token 训练规模下，Gated DeltaNet-2 在所有对比模型（Mamba-2、Gated DeltaNet、KDA、Mamba-3）中取得最强综合成绩。在 RULER 长上下文基准测试中，多键检索任务提升最为显著，在纯循环和混合架构两种设置下均超越全部基线。语言建模困惑度和常识推理任务相比前代方法 Gated DeltaNet 和 KDA 也持续提升。 At 1.3B parameters / 100B tokens, Gated DeltaNet-2 achieves the strongest overall results among all compared models (Mamba-2, Gated DeltaNet, KDA, Mamba-3). On RULER long-context benchmarks, it shows the most pronounced gains specifically in the multi-key retrieval setting, outperforming all baselines in both recurrent and hybrid configurations. It consistently improves across language modeling perplexity and commonsense reasoning tasks compared to its predecessors Gated DeltaNet and KDA.
🏁 结论 Gated DeltaNet-2 证明了在线性注意力中通过独立的逐通道门解耦擦除与写入操作，是一种简单而有效的提升压缩记忆编辑能力的方式。该模型在 13 亿参数规模的亚二次方序列模型中树立了新的最优水平，尤其在长上下文检索任务上表现突出。 Gated DeltaNet-2 proves that decoupling the erase and write operations in linear attention — via separate channel-wise gates — is a simple yet effective way to improve compressed memory editing. The model establishes a new state-of-the-art among sub-quadratic sequence models at the 1.3B scale, with especially strong long-context retrieval performance.

⚙️ Method · 方法详解

Gated DeltaNet-2 用两个独立的逐通道门替换了先前 delta-rule 模型中的单标量门：擦除门 b_t 控制键侧旧记忆的清除程度，写入门 w_t 控制值侧新内容的写入程度。这种解耦被整合进非对称的快速权重更新公式中。为支持高效训练，作者推导了将逐通道衰减吸收进擦除因子的分块 WY 表示，并设计了感知门控的反向传播以保持序列维度的并行性。该模型同时兼容纯循环和混合（循环+注意力）架构。 Gated DeltaNet-2 replaces the single scalar gate used in prior delta-rule models with two independent channel-wise gates: an erase gate b_t that controls how much of the old memory on the key side is removed, and a write gate w_t that controls how much new content is committed on the value side. This separation is absorbed into an asymmetric fast-weight update formulation. To support efficient training, the authors derive a chunkwise WY representation that incorporates channel-wise decay into the erase factors, and design a gate-aware backward pass that maintains parallelism across the sequence dimension. The model is compatible with both pure recurrent and hybrid (recurrent + attention) architectures.

\[\mathbf{S}_t = \mathbf{S}_{t-1} + \mathbf{w}_t \odot (\mathbf{v}_t - \mathbf{S}_{t-1}\mathbf{k}_t)^\top \mathbf{k}_t^\top - \mathbf{b}_t \odot \mathbf{S}_{t-1} \hat{\mathbf{k}}_t \hat{\mathbf{k}}_t^\top\]
💡 Gated Delta Rule-2 的状态更新公式：用写入门 w_t 控制新值的写入量，用擦除门 b_t 控制旧记忆的擦除量，两者独立作用于同一记忆状态 S_t。 The Gated Delta Rule-2 state update: write gate w_t independently controls how much new value is committed, while erase gate b_t independently controls how much old memory is removed — both operating channel-wise on the recurrent state S_t.
\[\text{KDA: } \mathbf{b}_t = \mathbf{w}_t = \beta_t \cdot \mathbf{1}, \quad \text{Gated DeltaNet: } \mathbf{b}_t = \mathbf{w}_t = \beta_t \cdot \mathbf{1},\ \lambda=1\]
💡 当擦除门和写入门退化为同一标量时，模型还原为 KDA；当衰减也退化（λ=1）时，还原为 Gated DeltaNet，展示了 Gated DeltaNet-2 对先前方法的统一覆盖关系。 When both gates collapse to the same scalar β_t, the model reduces to KDA; when channel-wise decay also collapses (λ=1), it reduces to Gated DeltaNet — showing Gated DeltaNet-2 as a strict generalization of both.

🌟 Why It Matters · 为什么重要

随着大语言模型处理越来越长的上下文，softmax 注意力的内存瓶颈推动了对高效线性替代方案的需求，Gated DeltaNet-2 通过更细粒度的解耦记忆控制推进了这一前沿。该工作同时为近期 delta-rule 架构提供了清晰的理论统一框架，为后续研究者提供了有原则可依的设计空间。 As LLMs scale to longer contexts, the memory bottleneck of softmax attention drives demand for efficient linear alternatives — Gated DeltaNet-2 advances this frontier by showing that fine-grained, decoupled memory control meaningfully improves both quality and retrieval. This work also provides a clean theoretical unification of recent delta-rule architectures, giving future researchers a principled design space to explore.

Post-Training is About States, Not Tokens: A State Distribution View of SFT, RL, and On-Policy Distillation 后训练的核心是状态而非词元：从状态分布视角看SFT、RL与在线蒸馏

🧑‍🔬 Dong Nie

图 Figure · Page 5 of paper

🎯 后训练效果取决于训练状态分布，而非仅仅是损失函数形式。
Post-training quality hinges on training state distribution, not just loss function choice.
维度 中文 English
❓ 待解决
的问题		 大语言模型的后训练方法（SFT、RL、蒸馏）通常只从损失函数角度分析，却忽视了一个关键因素：模型究竟在哪些"状态"上被训练。不同方法让模型接触到本质上不同的训练状态分布，而这一区别在现有研究中基本被忽视。 LLM post-training methods (SFT, RL, distillation) are almost always analyzed through the lens of their loss functions. However, this view overlooks a critical factor: which states (prompt + generated prefix pairs) the model is actually trained on. Different methods expose the model to fundamentally different distributions of training states, and this distinction is largely ignored in existing analysis.
💡 解决
的亮点		 • 全新框架：将SFT、RL、在线蒸馏（OPD）统一在"状态分布塑造"视角下，将分析重心从损失函数形式转移到监督信号施加的位置。 • 核心洞察：在线方法（RL、OPD）在当前学习者自身生成的状态上训练，具备自我修正探索能力；而SFT被锁定在静态数据集状态上。 • 惊人发现：用一个已经退化的SFT教师进行在线蒸馏，学生模型在多个基准上仍能超越该教师，说明状态局部性比教师质量更重要。 • Novel framing: Reinterprets all post-training methods (SFT, RL, OPD) under a unified "state distribution shaping" lens, shifting focus from loss form to where supervision is applied. • Key insight: On-policy methods (RL, OPD) train on states generated by the current learner, enabling self-correcting exploration, while SFT is locked to static dataset states. • Surprising finding: On-policy distillation from a *degraded* SFT teacher can surpass the teacher itself on multiple benchmarks, suggesting state locality matters more than teacher quality.
🧪 实验
内容		 实验以Qwen3-0.6B-Base为基础模型，以GSM8K为目标训练任务，以TruthfulQA和MMLU为保留评估基准，衡量灾难性遗忘程度。共设置四种实验条件：温和SFT、过度SFT（压力测试）、以退化SFT模型为教师的在线蒸馏（OPD），以及轻量级在线RL。实验采用内部受控对比设计，以隔离状态分布的影响，不引入外部基线。 Experiments use Qwen3-0.6B-Base as the base model, trained on GSM8K as the target task, with TruthfulQA and MMLU as retention benchmarks to measure catastrophic forgetting. Four conditions are compared: mild SFT, stress (heavy) SFT, on-policy distillation using the stress-SFT model as a degraded teacher, and lightweight on-policy RL. No external baselines from other papers are used; the comparison is internal and controlled to isolate the effect of state distribution.
📊 实验
结果		 温和SFT在几乎不损失保留能力的情况下提升了GSM8K表现。过度SFT导致TruthfulQA和MMLU保留基准上的显著下降。基于退化SFT教师的OPD在GSM8K、TruthfulQA和MMLU三项基准上均超越了该退化教师，尽管监督信号仅来源于该教师。轻量级在线RL在提升GSM8K准确率的同时，保留了TruthfulQA和MMLU的性能。（注：摘要中未披露具体数值，以方向性结论为主。） Mild SFT improves GSM8K performance with minimal retention loss. Stress SFT causes substantial drops on TruthfulQA and MMLU retention benchmarks. OPD using the degraded stress-SFT teacher surpasses that teacher on all three benchmarks: GSM8K, TruthfulQA, and MMLU — despite having no supervision source other than the degraded teacher. Lightweight on-policy RL improves GSM8K accuracy while preserving retention scores on TruthfulQA and MMLU. (Note: exact numerical figures are not reported in the abstract; results are described in qualitative-to-directional terms in the paper.)
🏁 结论 本文证明，训练状态的分布——而非仅仅是损失函数的形式——是后训练效果（包括任务提升和灾难性遗忘）的核心驱动因素。以状态为中心的统一框架将SFT、RL和在线蒸馏纳入同一分析体系，为设计更鲁棒的后训练流程提供了可操作的指导原则。 This paper establishes that the distribution of training states — not just the form of the loss function — is a primary driver of post-training outcomes, including task improvement and catastrophic forgetting. The state-centric framework unifies SFT, RL, and on-policy distillation and offers actionable guidance for designing more robust post-training pipelines.

⚙️ Method · 方法详解

本文将后训练形式化为"状态分布塑造"问题。对于自回归策略，"状态"定义为提示词与已生成前缀的拼接。SFT将状态分布固定在静态数据集上，而RL和在线蒸馏（OPD）则持续将状态分布更新为当前学习者自身的输出。作者基于Qwen3-0.6B-Base进行受控小规模实验，比较温和SFT与过度SFT、使用退化教师的OPD以及轻量级在线RL，在目标任务和保留任务上分别评估。 The paper formalizes post-training as a state-distribution shaping problem. For an autoregressive policy, a "state" is defined as a prompt concatenated with a generated prefix. SFT fixes the state distribution to a static dataset, while RL and on-policy distillation (OPD) continuously update the state distribution to match the current learner's own outputs. The authors run controlled small-scale experiments using Qwen3-0.6B-Base, comparing mild vs. stress SFT runs, OPD from a degraded teacher, and lightweight on-policy RL, all evaluated on target and retention benchmarks.

🌟 Why It Matters · 为什么重要

本文挑战了仅从损失目标角度比较后训练方法的主流范式，为在线方法通常泛化更好、遗忘更少提供了更本质的解释。对于设计大模型训练流程的从业者来说，状态分布视角提供了一个直观且可操作的工具，用于在不改变损失函数的前提下诊断和缓解灾难性遗忘。 This reframing challenges the dominant paradigm of comparing post-training methods purely by their loss objectives, offering a more principled explanation for why on-policy methods tend to generalize better and forget less. For practitioners designing LLM training pipelines, the state-distribution lens provides a concrete, intuitive handle for diagnosing and mitigating catastrophic forgetting without changing loss functions.

Vector Policy Optimization: Training for Diversity Improves Test-Time Search 向量策略优化：多样性训练提升测试时搜索效果

🧑‍🔬 Ryan Bahlous-Boldi, Isha Puri, Idan Shenfeld, Akarsh Kumar et al.

图 Figure · Page 4 of paper

🎯 用向量奖励训练多样性，大幅提升LLM推理时搜索能力
VPO trains LLMs for solution diversity by optimizing vector rewards, boosting inference-time search.
维度 中文 English
❓ 待解决
的问题		 当前主流的LLM强化学习后训练方法（如GRPO）只优化一个标量奖励，导致模型输出的解答分布熵值很低、缺乏多样性。而在AlphaEvolve等推理时搜索框架中，系统需要模型产出覆盖不同权衡取向的多样候选解，标量训练范式在这里严重失效。目前业界还没有一种通用方法能让LLM主动学会生成多样化解答。 Modern LLMs are trained with scalar reward signals (e.g., GRPO), which causes them to collapse toward low-diversity, repetitive outputs. When these models are used inside inference-time search pipelines—like AlphaEvolve—that need diverse candidate solutions to explore different trade-offs, this lack of diversity becomes a critical bottleneck. There is no standard way to train LLMs to proactively generate varied solutions suited for downstream search.
💡 解决
的亮点		 • VPO将GRPO中的标量优势估计器替换为**向量值**优势，让每个输出解专门适应不同的奖励维度组合。 • 核心洞察：现实中的奖励本来就是多维的（如代码生成中每个测试用例的通过情况、多个用户画像），VPO直接利用这一天然结构。 • VPO几乎是即插即用的替换方案——改动极小，却从根本上将训练目标从"找到最优解"转变为"覆盖多样的奖励空间"。 • VPO replaces the scalar advantage estimator in GRPO with a **vector-valued** advantage, allowing each solution to specialize toward different reward dimensions simultaneously. • The key insight is that many real-world rewards are naturally multi-dimensional (e.g., per-test-case correctness in code, multiple user personas), and VPO explicitly exploits this structure. • VPO is a near drop-in replacement—minimal architectural change, yet it fundamentally shifts the training objective from "find one best answer" to "cover the diverse reward space."
🧪 实验
内容		 实验涵盖四个任务：竞赛编程（pass@k / best@k）、数学解题，以及受AlphaEvolve启发的进化搜索场景。基线模型包括标准GRPO和其他标量RL变体。评估指标为不同搜索预算k下的pass@k和best@k，以及在进化搜索任务中可解问题数量。实验系统性地改变测试时搜索预算，以考察多样性收益随算力增加的变化规律。 Experiments were conducted across four tasks: competitive coding (pass@k / best@k on coding benchmarks), math problem solving, and evolutionary search scenarios inspired by AlphaEvolve. Baselines included standard GRPO and other scalar RL variants. Evaluation metrics were pass@k, best@k at varying search budgets (k), and success rate on problems solvable only via evolutionary search. The test-time search budget was systematically varied to examine how diversity benefits scale with compute.
📊 实验
结果		 在全部四个任务的pass@k和best@k指标上，VPO均与最强标量RL基线（GRPO）持平或超越，且随着搜索预算k增大，性能差距持续扩大。在进化搜索任务上，VPO训练的模型能解决GRPO模型完全无法解决的问题，实现了能力上的质变突破。多样性优势在大k时最为显著，验证了VPO训练目标与推理时搜索覆盖度之间的直接关联。 VPO matches or outperforms the strongest scalar RL baselines (GRPO) on pass@k and best@k across all four tasks, with performance gaps widening as search budget k increases. For evolutionary search tasks, VPO-trained models unlock problems that GRPO-trained models cannot solve at all—representing a qualitative jump in capability. The diversity advantage is most pronounced at large k, confirming that VPO's training objective directly translates to better inference-time search coverage.
🏁 结论 VPO证明：通过向量值奖励显式训练LLM的多目标多样性（作为GRPO的即插即用替换），能够大幅提升测试时搜索性能，尤其在更大推理预算下优势更加突出，并能解锁标量训练模型完全无法攻克的问题。 VPO proves that explicitly training LLMs for multi-objective diversity—using vector-valued rewards as a drop-in replacement for scalar GRPO—substantially improves test-time search performance, especially at larger inference budgets, and unlocks problems inaccessible to scalar-trained models.

⚙️ Method · 方法详解

VPO将LLM策略优化重新定义为多目标问题。与GRPO计算单一标量优势不同，VPO为每次推出（rollout）计算一个**向量优势**：将每个解在所有奖励维度上的表现与基准（如组内均值）对比，鼓励同一批次中不同的解专门适应不同的奖励权衡方向。这个向量优势替代GRPO中的标量优势来计算策略梯度更新。训练完成后，该策略在推理时自然产出覆盖奖励帕累托前沿的多样化解集，与best-of-k、pass@k及进化搜索等流程高度兼容。 VPO reframes LLM policy optimization as a multi-objective problem. Instead of computing a single scalar advantage for each rollout, VPO computes a vector advantage by comparing each solution's performance across all reward dimensions against a reference (e.g., group mean), encouraging different solutions within a batch to specialize in different reward trade-offs. This vector advantage is then used in place of the standard GRPO advantage to update the policy gradient. The result is a policy that, at inference time, naturally produces a diverse set of solutions that span the reward Pareto frontier, making it far more compatible with best-of-k, pass@k, and evolutionary search procedures.

📐 Key Formulas · 关键公式

\[\hat{A}_i^{\text{VPO}} = \mathbf{w}_i^\top \mathbf{r}_i - \frac{1}{G}\sum_{j=1}^{G} \mathbf{w}_j^\top \mathbf{r}_j\]
💡 VPO的向量优势估计器：将每个解的向量奖励 r_i 与一组权重向量 w_i 内积后，减去同批次所有解的加权奖励均值，从而鼓励不同解专攻不同的奖励维度组合。 VPO's vector advantage estimator: the dot product of solution i's reward vector r_i with its assigned weight vector w_i, minus the group mean of all weighted rewards. This encourages each solution to specialize toward a different trade-off in the multi-dimensional reward space.

🌟 Why It Matters · 为什么重要

随着推理时扩展（best-of-k、进化搜索、验证器引导搜索）成为部署强大AI系统的主流范式，训练目标也必须随之进化——VPO直接弥合了这一对齐缺口。本文表明，面向多样性的后训练可能需要成为默认标准，将从根本上重塑学界对LLM强化学习微调的认知框架。 As inference-time scaling (best-of-k, evolutionary search, verifier-guided search) becomes the dominant paradigm for deploying capable AI systems, the training objective must evolve to match—VPO directly addresses this alignment gap. This work signals that diversity-aware post-training may need to become the default standard, reshaping how the field thinks about RL fine-tuning for LLMs.

This paper presents a multi-agent reinforcement learning (MARL) approach for training autonomous racing systems to compete safely and agilely on tracks shared with other cars in the real world. Unlike prior single-agent methods that treat other vehicles as background noise, this approach teaches agents to coordinate with other participants, overcoming the brittleness of autonomous systems in dynamic shared environments. The resulting system achieves superhuman performance in both safety and agility.
🏁 结论：多智能体强化学习让自主赛车超越人类并保障安全
Multi-agent reinforcement learning enables autonomous racing cars to surpass human-level performance in both speed and safety in shared real-world environments.

This paper addresses the redundancy found in Markov Decision Processes (MDPs) arising from symmetries and shared structure among state-goal pairs in offline Goal-Conditioned Reinforcement Learning (GCRL). The authors show that hierarchical policies not only reduce planning horizon through temporal abstraction, but also enable state abstraction to eliminate redundancy and improve learning efficiency. Both theoretical analysis and empirical results support the benefits of abstraction in offline GCRL settings.

This paper addresses two key challenges in the Flexible Job Shop Scheduling Problem (FJSP): unpredictable job arrivals and combinatorial complexity that makes traditional solvers impractical. The authors propose an event-driven deep reinforcement learning approach that dynamically responds to new job arrivals and makes real-time scheduling decisions. The learned policy can handle uncertainty in production environments without relying on mixed-integer linear programming solvers.
