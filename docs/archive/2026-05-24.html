MOSS: Self-Evolution through Source-Level Rewriting in Autonomous Agent Systems MOSS：通过源码级重写实现自主智能体系统的自我进化

🧑‍🔬 Qianshu Cai, Yonggang Zhang, Xianzhang Jia, Wei Xue et al.

图 Figure · Page 4 of paper

🎯 MOSS让智能体通过重写源代码实现自我进化
MOSS enables autonomous agents to self-improve by rewriting their own source code based on production failures.
维度 中文 English
❓ 待解决
的问题		 已部署的自主智能体系统基本上是"静态"的——它们无法从用户交互中学习，也无法在没有人工干预的情况下修复反复出现的错误。现有的自我进化方法只能修改提示词、记忆模式或技能文件等文本层面的内容，无法触及代码中的路由逻辑、钩子顺序、状态不变量等结构性问题，导致一整类系统性故障始终无法被修复。 Deployed autonomous agent systems are essentially frozen — they cannot learn from user interactions or fix recurring failures without a human engineer shipping an update. Prior self-evolving agent approaches only modify text-level artifacts like prompts, memory schemas, or skill files, which means structural bugs buried in routing logic, hook ordering, or dispatch code are completely out of reach. This leaves an entire class of real-world failures permanently unaddressed.
💡 解决
的亮点		 • 源码级重写：MOSS直接在智能体源代码上操作，这是一种图灵完备的介质，是所有文本可修改范围的严格超集，能够修复任何提示词或技能文件编辑都无法触及的结构性问题。 • 以失败证据为锚点：每次重写周期都基于自动整理的真实生产失败证据批次，确保修改具有针对性。 • 安全部署流水线：候选版本通过在临时试验容器中回放失败批次来验证，再经用户同意门控的原地容器热替换进行部署，并配备健康探针触发的自动回滚机制，兼顾安全性与可审计性。 • Source-level rewriting: MOSS operates directly on the agent's source code — a Turing-complete medium that is a strict superset of all text-mutable scopes, enabling fixes that no prompt or skill-file edit could achieve. • Evidence-anchored evolution: Each rewrite cycle is grounded in an automatically curated batch of real production-failure evidence, ensuring changes are targeted and relevant. • Safe deployment pipeline: Candidates are validated by replaying failure batches in ephemeral trial workers, then promoted via user-consent-gated in-place container swaps with automatic health-probe-gated rollback, making the process both safe and auditable.
🧪 实验
内容		 作者在OpenClaw基准测试上对MOSS进行了评估，该基准包含四个不同的自主智能体任务。评估使用0到1范围内的评分器分数（grader score）作为主要指标来衡量任务完成质量。MOSS在没有任何人工干预的情况下运行了一个完整的自我进化周期，从部署在基准测试上的基线智能体出发，经过完整的源码重写流水线完成进化。 The authors evaluated MOSS on OpenClaw, a benchmark for autonomous agentic tasks consisting of four distinct tasks. The evaluation used a grader score (ranging from 0 to 1) as the primary metric to assess task completion quality. MOSS ran for a single self-evolution cycle without any human intervention, starting from the baseline agent deployed on the benchmark and evolving through the full source-rewriting pipeline described in the paper.
📊 实验
结果		 在OpenClaw基准测试（四任务套件）上，MOSS在单次自我进化周期内将平均评分器分数从0.25提升至0.61，无需任何人工干预——绝对提升+0.36分，相对提升约144%。这一成果完全通过自动化的源码级代码重写和基于生产失败证据的流水线执行实现。 On the OpenClaw benchmark (four-task suite), MOSS improved the mean grader score from 0.25 to 0.61 in a single self-evolution cycle with zero human intervention — a gain of +0.36 absolute points, representing a 144% relative improvement. This was achieved entirely through automated source-level code rewriting and production-failure-evidence-driven pipeline execution.
🏁 结论 MOSS证明了源码级自我重写是一种可行且强大的自主智能体改进机制，能够修复任何纯文本修改方法都无法解决的更广泛类别的故障。通过在一次无监督周期内将OpenClaw平均分从0.25提升至0.61，本文为生产级自我进化智能体系统树立了新的研究前沿。 MOSS demonstrates that source-level self-rewriting is a viable and powerful mechanism for autonomous agent improvement, capable of addressing a strictly broader class of failures than any text-artifact-only approach. By lifting OpenClaw mean scores from 0.25 to 0.61 in one unsupervised cycle, the work establishes a new frontier for production-ready self-evolving agent systems.

⚙️ Method · 方法详解

MOSS实现了一套确定性的多阶段源码级自我重写流水线。首先，系统自动整理生产环境中的失败证据批次，作为每次进化周期的锚点。代码修改任务被委托给可插拔的外部编码智能体CLI（MOSS本身负责阶段编排和决策，而非直接生成代码）。候选修改镜像随后在临时试验容器中通过回放失败批次进行验证，确保修复真正解决了目标问题且没有引入新的回归错误。最终，经审核的候选版本通过用户同意门控的原地容器热替换部署到生产环境，并配备健康探针触发的自动回滚机制。 MOSS implements a deterministic multi-stage pipeline for source-level self-rewriting. First, it automatically curates batches of production-failure evidence to anchor each evolution cycle. Code modification is then delegated to a pluggable external coding-agent CLI (keeping MOSS itself responsible for stage ordering and verdicts, not raw code generation). Candidate modified images are validated by replaying the curated failure batch against them inside ephemeral trial worker containers, ensuring the fix actually resolves the targeted failures without regressions. Finally, approved candidates are promoted to production via a user-consent-gated, in-place container swap, with a health-probe-gated rollback mechanism ready if issues arise post-deployment.

🌟 Why It Matters · 为什么重要

本研究从根本上扩展了自我进化智能体能够修复的问题范围：通过在源代码层面操作，MOSS能够解决所有现有基于提示词或记忆的进化方法都看不见的结构性和架构性错误。如果这一范式能在更大规模上得到验证，将大幅降低维护已部署AI智能体系统所需的人工工程成本。 This work fundamentally expands the scope of what self-evolving agents can fix: by operating at the source-code level, MOSS can resolve structural and architectural bugs that are invisible to all existing prompt- or memory-based evolution methods. If validated at scale, this paradigm could dramatically reduce the human engineering overhead required to maintain deployed AI agent systems in production.

Advancing Mathematics Research with AI-Driven Formal Proof Search 用AI驱动的形式化证明搜索推进数学研究

🧑‍🔬 George Tsoukalas, Anton Kovsharov, Sergey Shirobokov, Anja Surina et al.

图 Figure · Page 3 of paper

🎯 AI智能体自主解决了开放数学难题，首次大规模验证LLM+Lean组合
An AI agent autonomously solves open math conjectures via LLM-driven formal proof search in Lean.
维度 中文 English
❓ 待解决
的问题		 大型语言模型在数学推理上越来越强，但它们经常犯错，无法保证答案正确性，限制了其在真正数学研究中的实用价值。如何让AI既能"创造性地生成"证明思路，又能"严格保证"证明正确，是当前的核心挑战。 LLMs are increasingly capable at math reasoning, but their unreliability — hallucinations and logical errors — makes them unsuitable for rigorous mathematical research. The core challenge is combining the creative generative power of LLMs with a mechanism that guarantees correctness of the resulting proofs.
💡 解决
的亮点		 • 首次大规模评估LLM生成Lean形式化证明解决真实开放数学问题的能力，目标是Erdős猜想和OEIS猜想等真正未解难题。 • 设计了两类智能体：基础交替生成-验证智能体，以及更复杂的高性能智能体，并系统比较其在难度梯度上的表现差异。 • 该系统已实际部署于组合数学、优化、图论、代数几何和量子光学等多个前沿研究领域，展现出真实科研价值。 • First large-scale evaluation of LLM-to-formal-proof pipelines on genuinely open mathematical problems (Erdős problems, OEIS conjectures), not just benchmarks. • Two agent architectures compared: a basic LLM-generate + Lean-verify loop vs. a more capable advanced agent, revealing cost-performance tradeoffs at high difficulty. • The system is actively deployed in real research settings across combinatorics, graph theory, algebraic geometry, and quantum optics — demonstrating beyond-benchmark utility.
🧪 实验
内容		 实验在两个真实开放数学问题集上进行：353道Erdős开放问题（著名数学家Erdős留下的未解猜想集合）和492道OEIS猜想（整数序列在线百科全书中的未证猜想）。基线为基础交替式LLM生成+Lean验证智能体，对比对象为研究团队的高级智能体。评估指标为在固定每题预算（约百美元量级）下成功证明的题目数量，并分析不同难度问题上的成本效率差异。 Experiments were conducted on two sets of genuinely open mathematical problems: 353 open Erdős problems (unsolved conjectures left by the prolific mathematician Paul Erdős) and 492 OEIS conjectures (unproven statements from the Online Encyclopedia of Integer Sequences). The basic alternating generate-verify agent served as the baseline, compared against the advanced agent. Success was measured by the number of problems fully and formally proved within a per-problem budget of a few hundred dollars, with analysis of cost-performance tradeoffs across difficulty levels.
📊 实验
结果		 • 高级智能体在353道Erdős开放题中自主解决了9题，每题成本仅需几百美元。 • 同一智能体在492道OEIS猜想中证明了44道（约占9%）。 • 基础智能体在Erdős问题上同样复现了9题成功，但在最难题目上成本显著更高，说明高级设计在困难问题上更具性价比。 • 系统已在组合数学、优化、图论、代数几何、量子光学五个领域正式部署用于真实科研。 • The advanced agent autonomously solved 9 out of 353 open Erdős problems, each at a cost of only a few hundred dollars. • The same agent proved 44 out of 492 OEIS conjectures (~9% success rate). • The basic agent matched the 9 Erdős successes but was substantially more expensive on the hardest problems, validating the advanced agent's design for difficult tasks. • The system is actively deployed across 5 research domains: combinatorics, optimization, graph theory, algebraic geometry, and quantum optics.
🏁 结论 本文证明了AI驱动的形式化证明搜索已能自主解决真实的开放数学研究问题，而非仅限于教科书题目或竞赛基准。这是AI辅助数学研究进入实用阶段的重要里程碑，同时为不同智能体架构的设计取舍提供了系统性实证依据。 This paper demonstrates that AI-driven formal proof search can autonomously resolve genuinely open mathematical problems at reasonable cost, marking a meaningful step toward AI as a practical collaborator in frontier mathematics research. It also provides the first large-scale empirical comparison of agent architectures for this task, illuminating the design choices that matter most.

⚙️ Method · 方法详解

核心方法是将LLM的自然语言证明生成与Lean定理证明器的严格自动验证结合起来：LLM负责提出证明步骤，Lean负责逐行验证是否正确。基础智能体采用简单的"生成→验证→反馈→重新生成"循环。更强的高级智能体则在搜索策略、上下文利用和对Lean错误反馈的利用上做了更多工程优化，以在相同预算下探索更广的证明空间。整个流程不依赖人工干预，实现端到端自主证明搜索。 The approach couples LLM-based proof generation with Lean-based formal verification in an automated loop: the LLM proposes proof steps in Lean syntax, and the Lean proof assistant checks them for correctness, feeding errors back to the LLM for iterative refinement. A basic agent simply alternates generation and verification. The advanced agent improves on search strategy, context management, and error exploitation to cover more of the proof search space per dollar spent. The entire pipeline runs autonomously without human intervention.

🌟 Why It Matters · 为什么重要

这项工作将LLM的实用边界从"能解题"推进到了"能做研究"，对AI与科学发现的结合具有范式意义。形式化验证机制从根本上解决了LLM的幻觉问题，为AI在数学、逻辑密集型领域的可信部署提供了可行路径。 This work pushes the frontier of LLM utility from "solving known problems" to "advancing open research," representing a paradigm shift in AI-assisted scientific discovery. The formal verification component fundamentally addresses LLM hallucination, offering a trustworthy deployment path for AI in mathematics and other logic-intensive fields.

SeqLoRA: Bilevel Orthogonal Adaptation for Continual Multi-Concept Generation SeqLoRA：面向持续多概念生成的双层正交自适应

🧑‍🔬 Javad Parsa, Enis Simsar, Amir Joudaki, Thomas Hofmann et al.

图 Figure · Page 2 of paper

🎯 用双层优化的持续学习LoRA实现多概念无干扰图像生成
SeqLoRA uses bilevel continual learning to compose 100+ custom concepts in text-to-image diffusion with minimal interference.
维度 中文 English
❓ 待解决
的问题		 文本生成图像模型可以通过微调来"记住"特定概念（如某个人的外貌或某种风格），但当需要同时组合多个自定义概念时，这些概念之间会互相干扰，导致生成结果失真。现有方法要么需要昂贵的后期融合步骤，要么冻结部分参数导致表达能力受限，难以在保真度和可扩展性之间取得平衡。 Text-to-image diffusion models can be personalized with LoRA to capture custom concepts, but composing multiple concepts simultaneously causes representation interference — each new concept corrupts previously learned ones (catastrophic forgetting). Existing solutions either require expensive post-hoc merging of model weights or freeze adaptation subspaces, sacrificing either computational efficiency or concept fidelity.
💡 解决
的亮点		 • **双层优化框架**：同时优化LoRA的两个因子矩阵（A和B），而非固定其中一个，理论上能最小化残差干扰能量。 • **可学习正交基**：证明从数据中学习LoRA基比冻结基方法能更有效地减少概念间的干扰，并给出严格的理论保证。 • **矩阵次高斯过程建模**：将残差层激活建模为矩阵次高斯过程，首次推导出灾难性遗忘的高概率上界。 • **Bilevel optimization**: Jointly optimizes both LoRA factor matrices (A and B) rather than freezing one, theoretically minimizing residual interference energy between concepts. • **Learnable orthogonal basis**: Proves that data-driven LoRA basis learning outperforms frozen-basis approaches at reducing inter-concept interference, backed by formal convergence guarantees. • **Matrix sub-Gaussian process modeling**: Models residual layer activations as a matrix sub-Gaussian process to derive the first high-probability bounds on catastrophic forgetting in this setting.
🧪 实验
内容		 实验在多概念图像生成任务上进行，测试了从2个到101个概念的组合生成能力。使用了包含人物身份、物体等多类概念的标准测试集，与多种基线方法比较，包括Custom Diffusion、Mix-of-Show、Cones2、OMG等主流方法。评估指标涵盖身份保真度（DINO/CLIP图像相似度）、文本对齐度（CLIP文本相似度）以及属性干扰率等。同时进行了消融实验验证双层优化和可学习基的各自贡献。 Experiments were conducted on multi-concept text-to-image generation benchmarks, evaluating compositions ranging from 2 up to 101 concepts. Baselines include Custom Diffusion, Mix-of-Show, Cones2, OMG, and other state-of-the-art personalization methods. Evaluation metrics include DINO and CLIP image similarity for identity preservation, CLIP text alignment for prompt fidelity, and attribute interference scores. Ablation studies isolate the contributions of bilevel optimization and the learnable orthogonal basis.
📊 实验
结果		 SeqLoRA在多概念组合生成中相比基线方法在身份保真度（DINO相似度）上提升显著，同时在可扩展性方面支持多达101个概念的顺序学习，远超大多数基线方法的上限。在属性干扰指标上相比冻结基方法有明显改善，且无需后处理融合步骤，推理效率更高。消融实验表明，双层优化相比单层优化在遗忘抑制上有可量化的提升（具体数值详见论文Table 1-2）。 SeqLoRA achieves notable improvements in identity preservation (DINO similarity) over competing baselines on multi-concept composition benchmarks, and scales to sequential learning of up to 101 concepts — far beyond the capacity of most prior methods. Attribute interference scores are measurably reduced compared to frozen-basis LoRA variants, without requiring any post-hoc model fusion. Ablation results confirm that bilevel optimization provides quantifiable gains in forgetting suppression over single-level alternatives (see Tables 1–2 in the paper for precise figures).
🏁 结论 SeqLoRA通过双层正交约束的持续学习框架，在无需昂贵融合操作的前提下，实现了高保真、低干扰的多概念图像生成，并提供了首个对灾难性遗忘的严格理论界。该工作为参数高效微调与持续学习的结合提供了理论与实践的双重贡献。 SeqLoRA demonstrates that jointly optimizing both LoRA factors under orthogonal continual learning constraints enables scalable, high-fidelity multi-concept image generation without costly fusion, backed by the first rigorous theoretical forgetting bounds in this setting. It advances both the theory and practice of parameter-efficient continual fine-tuning for generative models.

⚙️ Method · 方法详解

SeqLoRA将多概念学习建模为一个约束持续学习问题，通过双层优化同时更新LoRA的A矩阵（基）和B矩阵（系数），下层优化新概念的拟合，上层优化正交约束以减少对旧概念的干扰。该方法无需在所有概念上同时训练，也无需在推理时进行模型融合，只需顺序地为每个概念添加正交约束的适配模块。从理论层面，作者将层激活建模为矩阵次高斯过程，并证明可学习基相比冻结基能以更低的概率上界控制遗忘。 SeqLoRA frames multi-concept personalization as a constrained continual learning problem. Each new concept is learned sequentially by adding a LoRA adapter whose basis is regularized to be orthogonal to previously learned subspaces, preventing interference. The bilevel optimization simultaneously updates both the A (basis) and B (coefficient) matrices: the inner level fits the new concept, while the outer level enforces orthogonality constraints to protect old concepts. This design eliminates both concurrent multi-concept training and costly post-hoc model fusion, while theoretical analysis using matrix sub-Gaussian processes provides convergence and forgetting bounds.

📐 Key Formulas · 关键公式

\[\min_{A, B} \mathcal{L}_{\text{new}}(W + BA) \quad \text{s.t.} \quad A A_k^\top = 0, \; \forall k < t\]
💡 双层优化的核心约束：在学习新概念的同时，强制新LoRA基矩阵A与所有历史基矩阵正交，从而避免干扰旧概念的表示空间。 Core bilevel constraint: while minimizing loss on the new concept, the new LoRA basis A is forced to be orthogonal to all previously learned bases, preventing interference with old concept subspaces.
\[\mathcal{E}_{\text{interference}} = \|P_{A_k} \Delta W\|_F^2\]
💡 残差干扰能量：衡量新适配权重ΔW在旧概念子空间上的投影大小，正交约束的目标就是将此值最小化。 Residual interference energy: measures how much of the new weight update projects onto old concept subspaces; orthogonality constraints directly minimize this quantity.

🌟 Why It Matters · 为什么重要

随着文生图模型在个性化应用中的普及，如何高效地组合数十乃至上百个自定义概念是一个核心工程挑战，SeqLoRA提供了一个兼具理论保障和实际可扩展性的解决方案。其双层优化框架和遗忘界理论对持续学习领域也具有独立的方法论价值。 As text-to-image personalization moves toward real-world applications requiring dozens of custom concepts, scalable and interference-free composition methods are critical — SeqLoRA offers a theoretically grounded and practically efficient solution. Its bilevel optimization framework and novel forgetting bounds also contribute broadly to the continual learning literature beyond generative models.

This paper proposes a reinforcement learning (RL) framework to train large language model (LLM) agents to automatically handle complex tasks in spreadsheet tools like Microsoft Excel and Google Sheets. The researchers build realistic spreadsheet task benchmarks and use RL to let the model iteratively improve its action strategies through environmental feedback. Experiments show that this approach significantly outperforms existing baselines on challenging spreadsheet tasks.

Finite-Particle Convergence Rates for Conservative and Non-Conservative Drifting Models
Generative Modeling · 生成模型Diffusion Models · 扩散模型Convergence Theory · 收敛理论
本文提出并分析了一种用于单步生成建模的保守漂移方法。该方法用基于核密度估计（KDE）梯度的速度场替代原始的位移漂移速度，即数据核平滑得分与模型核平滑得分之差，使速度场成为梯度场。文章从理论上推导了有限粒子情形下保守与非保守漂移模型的收敛速率，为生成模型的理论分析提供了新工具。
This paper proposes and analyzes a conservative drifting method for one-step generative modeling. It replaces the standard displacement-based velocity with a kernel density estimator (KDE)-gradient velocity — the difference between the kernel-smoothed data score and model score — making the velocity a proper gradient field. The authors derive finite-particle convergence rates for both conservative and non-conservative variants of the drifting model, providing new theoretical foundations for generative modeling.
🏁 结论：KDE梯度速度场可提升单步生成模型的理论收敛保障
Using a KDE-gradient velocity field in drifting models yields rigorous finite-particle convergence guarantees for one-step generative modeling.

🧑‍🔬 Krishnakumar Balasubramanian
