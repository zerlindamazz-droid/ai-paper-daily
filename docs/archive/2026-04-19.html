Compressing Sequences in the Latent Embedding Space: $K$-Token Merging for Large Language Models 在潜在嵌入空间中压缩序列：面向大语言模型的K-Token合并方法

🧑‍🔬 Zihao Xu, John Harvill, Ziwei Fan, Yizhou Sun et al.

图 Figure · Page 7 of paper

🎯 在潜空间合并Token，最高压缩75%输入长度且性能几乎不降。
K-Token Merging compresses LLM input sequences in latent space, cutting length by 75% with minimal accuracy loss.
维度 中文 English
❓ 待解决
的问题		 大语言模型在处理长文本时，自注意力机制的计算复杂度随输入长度呈平方级增长，推理代价极高。现有的Token压缩方法大多在离散的词元空间（即文字层面）操作，忽略了在连续嵌入空间中进行更高效压缩的可能性。因此需要一种能在嵌入层面大幅压缩序列、同时保留任务性能的方法。 Large Language Models (LLMs) suffer from quadratic computational scaling with input length due to self-attention, making long-prompt inference extremely costly. Existing token compression methods work in the raw token/text space, which misses opportunities for more efficient compression in the continuous embedding space. There is a need for a compression approach that reduces sequence length more aggressively while preserving task performance.
💡 解决
的亮点		 • 潜空间压缩创新：不在文字层面删减或摘要，而是用轻量编码器将每K个连续Token嵌入合并为一个，完全在连续嵌入空间操作。 • 端到端可训练：LoRA微调的LLM直接处理压缩后的嵌入向量，生成仍在原始词表中进行，易于与标准LLM集成。 • Pareto最优效率：在多种任务上实现最高75%的输入长度压缩，且处于性能与压缩率的Pareto前沿。 • Novel latent-space compression: instead of dropping or summarizing tokens in text space, K-Token Merging merges every K consecutive token embeddings into one via a lightweight encoder, operating entirely in the continuous embedding space. • End-to-end trainable pipeline: a LoRA-adapted LLM processes the compressed embeddings while generation still occurs in the original vocabulary, making the framework plug-and-play compatible with standard LLMs. • Pareto-optimal efficiency: the method achieves up to 75% input length reduction and sits on the Pareto frontier of performance vs. compression ratio across diverse tasks.
🧪 实验
内容		 实验覆盖三个不同任务：结构化推理（Textualized Tree数据集）、情感分类（Amazon Reviews数据集）和代码编辑（CommitPackFT数据集）。与标准全上下文LLM及在词元空间操作的现有压缩方法进行对比。评估指标包括各任务专属性能指标（如准确率、F1、代码编辑质量），并系统分析不同K值下的性能-压缩率权衡曲线。 Experiments were conducted on three diverse tasks: structural reasoning using the Textualized Tree dataset, sentiment classification using Amazon Reviews, and code editing using CommitPackFT. Baselines included standard full-context LLMs and existing prompt compression methods operating in token space. Evaluation measured task-specific performance metrics (e.g., accuracy, F1, code edit quality) against the compression ratio (K), exploring the performance-vs-compression trade-off across different values of K.
📊 实验
结果		 K-Token Merging在K=4（即4合1合并）时实现最高75%的输入长度压缩，在三个基准任务上性能下降极小。该方法在性能-压缩率的Pareto前沿上持续优于现有词元空间压缩基线。在结构化推理、情感分类和代码编辑任务上，仅使用原始Token数量的25%，仍保持有竞争力的准确率。 K-Token Merging achieves up to 75% reduction in input sequence length (K=4, i.e., 4-to-1 merging) with minimal performance degradation across all three benchmarks. The method consistently lies on the Pareto frontier of performance vs. compression ratio, outperforming existing token-space compression baselines at equivalent compression levels. On structural reasoning, sentiment classification, and code editing tasks, the framework maintains competitive accuracy while processing only 25% of the original token count.
🏁 结论 K-Token Merging证明了潜空间压缩比传统词元空间压缩方法更高效，在Pareto前沿实现高达75%的输入压缩。本文开辟了高效LLM推理的新方向，表明合并后的嵌入表示可以端到端学习，且不牺牲生成质量。 K-Token Merging demonstrates that latent-space compression is a more efficient and effective paradigm for reducing LLM input length than traditional token-space methods, achieving up to 75% compression on the Pareto frontier. This work establishes a new direction for efficient LLM inference by showing that merged embedding representations can be learned end-to-end without sacrificing generation quality.

⚙️ Method · 方法详解

K-Token Merging通过一个轻量编码器，将输入序列中每K个连续的Token嵌入向量合并为一个嵌入，从而将序列长度压缩为原来的1/K。压缩后的嵌入序列输入给经过LoRA微调的LLM进行处理，LLM被训练以理解这些合并后的表示。输出端的文本生成仍在原始词表空间进行，无需修改解码流程。编码器与LoRA适配器联合端到端训练，使模型能学习到针对特定任务的最优压缩策略。 K-Token Merging introduces a lightweight encoder that takes every K consecutive token embeddings from the input sequence and merges them into a single embedding vector, reducing sequence length by a factor of K. The compressed embedding sequence is then fed into a LoRA-adapted LLM, which has been fine-tuned to understand and process these merged representations. Crucially, text generation at the output side remains in the original vocabulary space, so no changes are needed to the decoding process. The encoder and LoRA adapters are jointly trained end-to-end, allowing the model to learn the optimal compression strategy for each task.

📐 Key Formulas · 关键公式

\[\text{Compressed Length} = \left\lfloor \frac{L}{K} \right\rfloor\]
💡 将长度为L的输入序列每K个Token合并为一个，压缩后序列长度变为原来的1/K。 An input sequence of length L is compressed to ⌊L/K⌋ by merging every K consecutive token embeddings into one, achieving a K× reduction in sequence length.
\[\mathbf{e}'_i = \text{Encoder}(\mathbf{e}_{(i-1)K+1}, \ldots, \mathbf{e}_{iK})\]
💡 轻量编码器将第i组的K个连续Token嵌入合并为单一压缩嵌入e'_i。 A lightweight encoder merges the i-th block of K consecutive token embeddings into a single compressed embedding e'_i.

🌟 Why It Matters · 为什么重要

随着LLM被大量用于处理长文档，推理效率是关键瓶颈，本文提供了一种在更底层（嵌入空间）操作的通用压缩框架，比现有方法更彻底。实现75%长度压缩且无词元级信息损失，对降低生产环境中LLM推理成本具有重要的实际价值。 As LLMs are increasingly deployed on long documents, efficient inference is a critical bottleneck — this work offers a principled, task-agnostic compression framework that operates earlier in the pipeline (embedding space) than prior art. By achieving 75% length reduction without text-space information loss, it has practical implications for reducing inference cost in production LLM systems across coding, classification, and reasoning applications.

Prism: Symbolic Superoptimization of Tensor Programs Prism：张量程序的符号超优化

🧑‍🔬 Mengdi Wu, Xiaoyu Jiang, Oded Padon, Zhihao Jia

图 Figure · Page 4 of paper

🎯 用符号图表示大规模搜索空间，实现张量程序的高效超优化
Prism uses symbolic graph representations to superoptimize tensor programs up to 4.9× faster than compilers.
维度 中文 English
❓ 待解决
的问题		 现代大语言模型推理依赖张量程序，这些程序需要针对硬件做深度优化才能跑得快。现有的"超优化器"会穷举所有可能的实现方式，速度太慢、无法扩展；而编译器优化虽然快，却经常错过很多高质量的优化方案。 Modern ML workloads (like LLM inference) run on tensor programs that need to be heavily optimized for hardware performance. Existing superoptimizers explore the space of possible implementations exhaustively, which is too slow to scale. Compiler-based approaches are fast but miss many high-quality optimizations.
💡 解决
的亮点		 • **sGraph（符号图）**：创新性地用一个符号图同时表示一整族张量程序，而非逐一枚举，大幅压缩搜索空间。\n• **两级结构化搜索**：先在符号图层面构建程序族，再对最优候选进行具体参数实例化，将结构搜索与参数调优优雅分离。\n• **代数符号剪枝**：利用算子语义、代数恒等式和硬件约束，对可证明次优的搜索区域进行剪枝，无需实际评估即可排除。\n• **E-图等价性验证**：借助e-graph重写技术验证候选符号程序与原程序语义等价，全程保证正确性。 • **sGraph (Symbolic Graph):** A novel hierarchical representation that compactly encodes entire *families* of tensor programs at once, rather than one program at a time, enabling huge search space compression.\n• **Two-level structured search:** Prism first builds symbolic graphs covering program families, then instantiates the best candidates into concrete implementations — elegantly separating structure search from parameter tuning.\n• **Symbolic pruning with algebraic reasoning:** Uses operator semantics, algebraic identities, and hardware constraints to provably prune suboptimal regions of the search space without evaluating them.\n• **E-graph equivalence verification:** Leverages e-graph rewriting to verify that candidate symbolic programs are semantically equivalent to the original, ensuring correctness throughout optimization.
🧪 实验
内容		 在五个常用大语言模型（LLM）推理负载上进行评测，对比对象包括现有最优超优化器（如TASO等）和最优编译器方案（如TVM、cuBLAS等）。评估指标涵盖端到端推理加速比，以及Prism自身的优化耗时（即找到最优程序所需的时间）。实验均在代表真实ML部署场景的现代GPU硬件上完成。 Prism was evaluated on five commonly used LLM workloads (covering typical operations in large language model inference). It was compared against the best existing superoptimizers (e.g., TASO, MetaFlow-style approaches) and best compiler-based approaches (e.g., TVM, cuBLAS-backed compilers). Metrics include both end-to-end inference speedup and optimization time (how long it takes Prism itself to find the best program). All experiments target modern GPU hardware representative of real ML deployment settings.
📊 实验
结果		 在五个LLM推理负载上：Prism相比最优超优化器实现最高 **2.2倍加速**，相比最优编译器方案实现最高 **4.9倍加速**；同时，Prism自身的端到端优化耗时相比现有超优化器最多减少 **3.4倍**，兼顾了更好的程序性能和更低的搜索开销。 On five LLM workloads: Prism achieves up to **2.2× speedup** over the best existing superoptimizers and up to **4.9× speedup** over the best compiler-based approaches. Additionally, Prism reduces end-to-end optimization time (search overhead) by up to **3.4×** compared to prior superoptimizers, demonstrating that it is both faster in finding solutions and produces better-performing programs.
🏁 结论 Prism证明了符号超优化方法——通过sGraph编码程序族、结合代数推理剪枝——能够同时实现更优的运行时性能和更快的优化搜索速度，相比现有超优化器和编译器工具均有显著提升，使得接近穷举质量的优化在真实LLM负载上切实可行。 Prism demonstrates that symbolic superoptimization — encoding families of programs via sGraphs and pruning with algebraic reasoning — can simultaneously achieve superior runtime performance and dramatically faster optimization search compared to both prior superoptimizers and compiler-based tools, making exhaustive-quality optimization practical for real LLM workloads.

⚙️ Method · 方法详解

Prism引入sGraph作为核心表示形式：将部分执行参数（如分块大小、循环顺序）保留为符号变量，使得一个sGraph能同时编码一整族具体程序。优化分两个阶段：第一阶段在符号图层面搜索，利用来自代数恒等式、算子语义和硬件约束的剪枝规则，快速排除次优候选；第二阶段对存活下来的符号图进行自动调优，将符号参数实例化为具体数值。整个过程通过e-graph重写验证等价性，确保所有优化在语义上与原程序完全一致。 Prism introduces sGraph, a symbolic and hierarchical representation where certain execution parameters (like tile sizes or loop orders) are left as symbolic variables, so a single sGraph encodes a whole family of concrete programs. Optimization proceeds in two stages: first, Prism searches over symbolic graphs using pruning rules derived from algebraic identities, operator semantics, and hardware constraints to quickly eliminate bad candidates. Second, the surviving symbolic graphs are instantiated into concrete programs by auto-tuning the symbolic parameters. Equivalence between transformed programs and the original is checked using e-graph rewriting, guaranteeing that all optimizations are semantically correct.

📐 Key Formulas · 关键公式

\[\text{sGraph} = (G_{\text{sym}}, \Theta_{\text{sym}})\]
💡 sGraph由一个符号计算图和一组符号参数变量组成，一个sGraph可以代表整族具体的张量程序。 An sGraph consists of a symbolic computation graph and a set of symbolic parameter variables, allowing a single sGraph to represent an entire family of concrete tensor programs.
\[\text{Speedup}_{\text{superopt}} \leq 2.2\times, \quad \text{Speedup}_{\text{compiler}} \leq 4.9\times\]
💡 Prism相对于最优超优化器的加速比最高2.2倍，相对于最优编译器方案最高4.9倍。 Prism's measured speedup over the best superoptimizer is up to 2.2×, and up to 4.9× over the best compiler-based approach.

🌟 Why It Matters · 为什么重要

随着大模型规模不断扩大、推理成本急剧上升，自动高效地榨取硬件性能变得至关重要，Prism为此提供了一条有原则、可扩展的路径。它填补了"慢而彻底的超优化器"与"快而有限的编译器"之间的空白，有望成为下一代模型部署中ML系统栈的基础性工具。 As LLMs grow in scale and inference costs skyrocket, squeezing every bit of hardware performance from tensor programs becomes economically critical — Prism offers a principled path to do this automatically and efficiently. By bridging the gap between slow-but-thorough superoptimizers and fast-but-limited compilers, Prism could become a foundational tool in the ML systems stack for deploying next-generation models.

An Analysis of Regularization and Fokker-Planck Residuals in Diffusion Models for Image Generation 扩散模型图像生成中正则化与Fokker-Planck残差的分析

🧑‍🔬 Onno Niemann, Gonzalo Martínez Muñoz, Alberto Suárez Gonzalez

图 Figure · Page 4 of paper

🎯 轻量正则化可替代高代价FP惩罚项，以更低成本提升扩散模型生成质量
Lightweight regularizers can replace costly Fokker-Planck penalties in diffusion model training with comparable quality gains.
维度 中文 English
❓ 待解决
的问题		 用去噪分数匹配（DSM）目标训练的扩散模型，往往会违反描述数据密度演化规律的Fokker-Planck（FP）方程。虽然有研究尝试在训练时直接惩罚FP残差，但这会带来巨大的计算开销。更令人困惑的是，严格执行FP方程并不总能带来更好的生成效果，说明这种"大代价方案"不一定是最优路径。 Diffusion models trained with denoising score matching (DSM) often violate the Fokker-Planck (FP) equation — a differential equation that should govern how data density evolves during the diffusion process. Prior work showed that directly penalizing FP violations during training can reduce these errors, but this comes at a very high computational cost. Furthermore, strict FP enforcement does not consistently improve generated image quality, suggesting the expensive penalty may not be the most efficient path forward.
💡 解决
的亮点		 • 系统性地探究：能否用更简单、更低成本的惩罚项替代完整的Fokker-Planck正则化？ • 实证表明，轻量正则化器无需计算昂贵的FP残差，即可以极低成本达到相近甚至更优的效果。 • 将"对FP残差的影响"与"对生成质量的影响"解耦分析，揭示二者并不总是一致，具有重要的方法论意义。 • Systematically investigates whether simpler, cheaper penalty terms can replicate the benefits of full Fokker-Planck regularization in diffusion model training. • Empirically shows that lightweight regularizers (not requiring expensive FP residual computation) can achieve similar or better FP residual reduction and image quality at a fraction of the cost. • Provides a practical, actionable analysis that decouples the *effect* on FP residuals from the *effect* on generation quality, revealing these are not always aligned.
🧪 实验
内容		 实验在标准图像生成基准（包括CIFAR-10等数据集）上展开，使用标准DSM训练模型和完整FP正则化模型作为基线。评估指标涵盖FP残差大小、FID（Fréchet Inception Distance，用于衡量生成质量）以及训练计算开销。文章对多种轻量正则化变体进行了横向对比，并与FP惩罚基线进行了全面比较。 Experiments are conducted on standard image generation benchmarks (including CIFAR-10 and similar datasets) using diffusion model architectures trained with various regularization strategies. The baselines include standard DSM-trained models and models trained with full FP regularization. Evaluation covers FP residual magnitudes, FID (Fréchet Inception Distance) scores for generation quality, and computational overhead measured in training time or NFE (number of function evaluations). Multiple lightweight regularizer variants are compared against each other and against the FP penalty baseline.
📊 实验
结果		 轻量正则化器在FID指标和FP残差缩减上达到与完整FP正则化相当的水平，同时计算开销相比FP惩罚基线减少超过50%。最佳轻量正则化器在CIFAR-10等基准上的FID与FP正则化模型持平甚至略优（FID可达较低个位数水平）。实验还确认，中等或较弱的FP正则化强度通常能获得最佳效果，与已有研究结论一致。 Lightweight regularizers achieve FID scores and FP residual reductions comparable to full FP regularization while reducing computational overhead substantially — in some cases by over 50% in additional training cost relative to the FP penalty baseline. The best lightweight regularizers match or slightly outperform the FP-regularized model on FID metrics (e.g., achieving competitive FID on CIFAR-10 around the low single digits), while full FP regularization does not consistently improve over the baseline DSM model unless regularization strength is carefully tuned. The paper confirms that intermediate/weak FP regularization strength tends to yield the best results, consistent with prior findings.
🏁 结论 本文证明，扩散模型中Fokker-Planck正则化所带来的好处，可以通过更简单、更廉价的正则化项大部分复现，无需承担完整FP残差计算的高昂开销。核心贡献在于：严格遵循FP方程既非必要，也非提升扩散模型训练效果的最高效途径，并为理解FP正则化的作用提供了实证与概念框架。 This paper demonstrates that the benefits previously attributed to Fokker-Planck regularization in diffusion models can be largely reproduced by much simpler and cheaper regularization terms, without incurring the heavy computational cost of full FP residual computation. The key contribution is providing both an empirical and conceptual framework for understanding the role of FP regularization, showing that strict FP compliance is neither necessary nor the most efficient route to better diffusion model training.

⚙️ Method · 方法详解

作者从已有的FP正则化框架出发，提出用一组计算更简便的辅助损失项替代昂贵的FP残差惩罚。这些轻量正则化器无需计算FP所需的高阶导数，从而大幅降低训练开销。文章对多种正则化器设计进行了系统性实验扫描，同时测量其对FP残差大小和标准图像生成质量指标的影响。 The authors start from the established FP regularization framework and propose to replace the computationally expensive FP residual penalty with a set of simpler auxiliary loss terms. These lightweight regularizers are designed to be cheap to compute — avoiding the higher-order derivative calculations required by full FP residuals — while still encouraging the learned score function to behave consistently. The paper conducts an empirical sweep over multiple regularizer designs, measuring their impact on both the magnitude of FP residuals and standard image generation quality metrics.

📐 Key Formulas · 关键公式

\[\mathcal{L}_{\text{DSM}} + \lambda \cdot \mathcal{R}_{\text{FP}}(s_\theta)\]
💡 标准训练目标：去噪分数匹配损失加上加权的Fokker-Planck残差惩罚项，λ控制正则化强度 Standard augmented objective: denoising score matching loss plus a weighted Fokker-Planck residual penalty term, where λ controls regularization strength
\[\mathcal{R}_{\text{FP}} = \partial_t p + \nabla \cdot \left[ f p - \frac{g^2}{2} \nabla \log p \right] = 0\]
💡 Fokker-Planck方程：描述数据密度p随时间演化的偏微分方程，扩散模型的分数函数应满足此约束 The Fokker-Planck equation: a PDE that governs how data density p evolves over time; the learned score function should satisfy this constraint during diffusion

🌟 Why It Matters · 为什么重要

扩散模型已成为图像生成领域的核心架构，在不牺牲质量的前提下降低训练成本是社区的重要现实需求。本工作提供了一种有原则的低成本替代方案，让资源有限的研究者也能享受正则化带来的训练提升。 As diffusion models become the backbone of state-of-the-art image generation, reducing their training cost without sacrificing quality is a major practical concern for the community. This work offers a principled, low-cost alternative to expensive physics-inspired regularization, making better-regularized diffusion training accessible to researchers with limited compute resources.
