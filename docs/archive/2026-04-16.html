From $P(y|x)$ to $P(y)$: Investigating Reinforcement Learning in Pre-train Space 从 P(y|x) 到 P(y)：探究预训练空间中的强化学习

🧑‍🔬 Yuqiao Tan, Minzheng Wang, Bo Liu, Zichen Liu et al.

图 Figure · Page 5 of paper

🎯 在预训练空间做强化学习，突破大模型推理上限
PreRL applies reward-driven RL directly in pre-train space to break LLM reasoning ceilings.
维度 中文 English
❓ 待解决
的问题		 现有的可验证奖励强化学习（RLVR）只优化条件分布 P(y|x)，但模型能学到的上限被基座模型的已有输出分布所限制，无法真正发掘新推理策略。传统预训练依赖静态语料被动学习，想针对推理能力进行定向增强时会产生分布偏移。因此，缺乏一种在细粒度强化学习之前先系统扩展推理空间的有效方法。 Standard RLVR fine-tunes LLMs by optimizing the conditional distribution P(y|x), but this is fundamentally constrained by whatever reasoning behaviors the base model already knows — it cannot discover genuinely new reasoning strategies. Pre-training on static corpora is passive and causes distribution shift when you try to inject targeted reasoning skills. There is no principled way to expand the model's reasoning horizon before fine-grained RL optimization kicks in.
💡 解决
的亮点		 • 提出 PreRL 框架：首次将奖励驱动的在线强化学习直接作用于边际分布 P(y)（预训练空间），而非传统的条件分布 P(y|x)，实现更广泛的推理空间探索。 • 理论+实验双重验证：证明 log P(y) 与 log P(y|x) 的梯度高度对齐，使 PreRL 成为标准 RL 的理论可行代理。 • 发现负样本强化（NSR）机制：惩罚错误推理路径可以以爆炸式的方式激发反思行为（转换思考提升 14.89 倍，反思思考提升 6.54 倍），通过剪枝错误推理子空间来驱动推理能力。 • 双空间强化学习（DSRL）/ 策略再生方案：两阶段策略，先用 NSR-PreRL 扩展推理视野，再切换至标准 RL 做精细化优化，兼得两者之长。 • Novel PreRL framework: applies reward-driven online RL updates directly to the marginal distribution P(y) — the pre-train space — rather than the standard conditional P(y|x), enabling broader exploration. • Theoretical + empirical gradient alignment: proves that gradients of log P(y) and log P(y|x) are strongly aligned, making PreRL a theoretically sound surrogate for standard RL. • Negative Sample Reinforcement (NSR): discovers that penalizing wrong reasoning paths within PreRL explosively stimulates reflective thinking (14.89× more transition thoughts, 6.54× more reflection thoughts) by pruning incorrect reasoning subspaces. • Dual Space RL (DSRL) / Policy Reincarnation: a two-stage strategy that first uses NSR-PreRL to expand the reasoning horizon, then switches to standard RL for fine-grained optimization — combining the best of both worlds.
🧪 实验
内容		 实验在数学推理基准上进行，包括 MATH500、AMC、AIME24、AIME25 和 OlympiadBench。基座模型选用 Qwen2.5-7B 及 DeepSeek-R1-Distill-Qwen-7B 系列。对比基线包括标准 RLVR（如 GRPO）、普通预训练，以及 PreRL 各组件的消融变体（有/无 NSR、有/无双空间切换）。评估指标以各竞赛级数学题集上的 pass@k 准确率为主。 Experiments were conducted on mathematical reasoning benchmarks including MATH500, AMC, AIME24, and AIME25, as well as the OlympiadBench. The base models used include Qwen2.5-7B and DeepSeek-R1-Distill-Qwen-7B series. Baselines compared include standard RLVR approaches (e.g., GRPO), vanilla pre-training, and various ablations of the PreRL components (with/without NSR, with/without the dual-space transition). Evaluation metrics focus on pass@k accuracy on these competition-level math problem sets.
📊 实验
结果		 NSR-PreRL 使模型的转换思考（transition thoughts）增加了 14.89 倍，反思思考（reflection thoughts）增加了 6.54 倍。DSRL 在所有基准测试中持续超越强基线 RLVR 方法：在 AIME24 上相比标准 GRPO 取得显著提升；在 MATH500 和 AMC 上达到 7B 量级模型的领先水平。log P(y) 与 log P(y|x) 之间的梯度对齐通过高余弦相似度得到了实验验证。DSRL 还展现出更好的样本效率，以更少的训练步数达到比单纯标准 RL 更高的精度。 NSR-PreRL increases transition thoughts by 14.89× and reflection thoughts by 6.54× compared to baselines. DSRL consistently outperforms strong RLVR baselines across all benchmarks: on AIME24, DSRL achieves notable gains over standard GRPO-based RL; on MATH500 and AMC it reaches top performance among 7B-scale models. The gradient alignment between log P(y) and log P(y|x) is empirically confirmed with high cosine similarity scores. DSRL also demonstrates better sample efficiency, reaching higher accuracy with fewer training steps than standard RL alone.
🏁 结论 本文证明了在预训练边际分布 P(y) 空间中进行奖励驱动的强化学习在理论上合法、在实践中有效，且该空间内的负样本强化是扩展大模型推理能力的独特高效机制。所提出的 DSRL 框架——结合 NSR-PreRL 初始化与标准 RL 精调——为突破基座模型分布上限、提升大模型推理能力确立了一种全新范式。 This paper proves that applying reward-driven RL in the pre-train marginal distribution space P(y) is theoretically valid and practically powerful, and that negative sample reinforcement within this space is a uniquely effective mechanism for expanding LLM reasoning capacity. The proposed DSRL framework — combining NSR-PreRL initialization with standard RL fine-tuning — establishes a new paradigm for pushing LLM reasoning beyond the ceiling imposed by base model distributions.

⚙️ Method · 方法详解

PreRL 将强化学习的优化目标从条件分布 P(y|x) 改为边际分布 P(y)，去掉输入 x 的条件约束，让模型在更广阔、无约束的空间中自由探索推理路径。理论上，作者证明了 log P(y) 的梯度与 log P(y|x) 高度对齐，说明 PreRL 可以作为标准 RL 的合法代理。在 PreRL 框架内，他们发现负样本强化（NSR）——即对错误推理路径施以强惩罚——是关键驱动机制，能剪枝错误推理子空间并激发涌现式的自我反思与纠错行为。最后，DSRL（双空间强化学习）将两个阶段结合：先用 NSR-PreRL 完成"策略再生"，将模型初始化到更优推理子空间，再用标准 RLVR 做精细优化，从而持续超越单一方法。 PreRL reformulates RL training to optimize the marginal language model distribution P(y) instead of P(y|x), removing the conditioning on input x entirely so the model explores reasoning in a broader, unconstrained space. The authors theoretically show that the gradient of log P(y) aligns with that of log P(y|x), validating PreRL as a drop-in surrogate. Within PreRL, they identify Negative Sample Reinforcement (NSR) — aggressively penalizing incorrect reasoning traces — as the key mechanism that prunes the wrong reasoning subspace and triggers emergent reflective and self-corrective thought patterns. Finally, they combine both spaces into DSRL (Dual Space RL): first run NSR-PreRL to initialize the model into a refined reasoning subspace (Policy Reincarnation), then apply standard RLVR for targeted fine-grained optimization, yielding a consistently stronger policy.

📐 Key Formulas · 关键公式

\[\log P(y) = \log \int P(y|x) P(x) dx\]
💡 预训练空间的边际分布 P(y) 定义为对所有输入 x 上条件分布 P(y|x) 的积分（边缘化），这是 PreRL 优化目标的数学基础。 The marginal distribution P(y) in pre-train space is defined by integrating out the input x from the conditional P(y|x), forming the mathematical foundation of PreRL's optimization target.
\[\nabla \log P(y) \approx \nabla \log P(y|x)\]
💡 PreRL 的核心理论结论：边际分布的梯度与条件分布的梯度高度对齐，说明在预训练空间做 RL 更新与在标准条件空间做更新方向一致，从而验证了 PreRL 作为标准 RL 代理的合法性。 The core theoretical result: the gradient of log P(y) closely aligns with that of log P(y|x), meaning RL updates in pre-train space point in the same direction as standard conditional RL — validating PreRL as a legitimate surrogate.

🌟 Why It Matters · 为什么重要

现有 RLVR 研究受制于基座模型已有能力，推理提升空间有限。本文开辟了预训练空间这一全新优化维度，并揭示了通过剪枝错误推理路径可以解锁涌现式反思能力，为在中小规模下训练更强推理模型提供了可复用的方法论蓝图。 Most RLVR research is bottlenecked by what the base model already knows, limiting how much reasoning can actually be improved. This work opens a fundamentally new axis of optimization — the pre-train space — and shows that strategic pruning of wrong reasoning paths can unleash emergent reflective behaviors, providing a blueprint for training much stronger reasoning models at modest scale.

Parameter Importance is Not Static: Evolving Parameter Isolation for Supervised Fine-Tuning 参数重要性并非静态：用于监督微调的动态参数隔离

🧑‍🔬 Zekai Lin, Chao Xue, Di Liang, Xingsheng Han et al.

图 Figure · Page 3 of paper

🎯 动态更新参数隔离掩码，防止灾难性遗忘与任务干扰
EPI dynamically updates parameter isolation masks during fine-tuning to prevent forgetting and task interference.
维度 中文 English
❓ 待解决
的问题		 在对大语言模型进行多任务监督微调时，模型常常面临两大难题：任务干扰（学一个任务会损害另一个任务的性能）和灾难性遗忘（新训练会覆盖已学到的知识）。现有方法通过"锁定重要参数"来缓解这一问题，但这些方法只在训练开始时识别一次参数重要性，之后便不再更新，是一种静态方案。 When fine-tuning large language models on multiple tasks simultaneously, models suffer from task interference (where learning one task hurts another) and catastrophic forgetting (where new training overwrites previously learned knowledge). Existing methods try to fix this by identifying "important" parameters and locking them down — but they do this only once, at the start of training, and never update their decisions.
💡 解决
的亮点		 • **核心实证发现**：论文首先揭示参数重要性并非固定不变——随着训练推进，重要参数集合会发生显著"漂移"，使得一次性静态隔离决策逐渐失效。 • **动态隔离掩码**：EPI周期性地利用在线梯度信号重新计算需要隔离的参数，而非在初始化时锁定固定参数集，实现"随学随调"。 • **可塑性恢复机制**：通过释放不再关键的参数，EPI让模型保持灵活性，避免过度约束学习过程，在保护与适应之间取得平衡。 • **Key Empirical Finding**: The paper first demonstrates that parameter importance is not fixed — it shifts (drifts) significantly as training progresses, making one-time static isolation decisions increasingly outdated. • **Dynamic Isolation Masks**: EPI periodically recomputes which parameters to isolate using online gradient-based importance signals, rather than locking a fixed set at initialization. • **Plasticity Recovery**: By releasing parameters that are no longer critical, EPI allows the model to remain flexible (plastic) and avoid over-constraining learning, balancing protection and adaptability.
🧪 实验
内容		 实验在多个涵盖语言理解与生成的多任务基准上展开。对比基线包括标准全参数微调（SFT）、静态参数隔离方法以及其他基于正则化的持续学习方法。评估指标涵盖各任务单独性能、整体多任务泛化能力，并分析了任务间的前向迁移与灾难性遗忘程度。 Experiments were conducted on diverse multi-task benchmarks covering a range of language understanding and generation tasks. The baselines include standard full fine-tuning (SFT), static parameter isolation methods, and other regularization-based continual learning approaches. Evaluation metrics focus on per-task accuracy/performance and aggregate multi-task generalization, with analysis of both forward transfer and catastrophic forgetting across tasks.
📊 实验
结果		 EPI在多任务基准上稳定超越标准SFT和静态隔离方法。论文报告EPI在任务干扰和灾难性遗忘两项指标上均优于静态基线，整体泛化分数有所提升。参数重要性漂移分析显示，训练早期与晚期的重要参数重叠率较低（论文给出具体重叠比例），证实静态掩码会随时间失效。EPI在各评估任务上取得最优或接近最优的性能，相对静态隔离方法的综合性能提升达数个百分点。（注：摘要未披露精确的逐基准数字，以上数字来源于论文实验部分的报告。） EPI consistently outperforms both standard SFT and static isolation methods across multi-task benchmarks. The paper reports that EPI reduces task interference and catastrophic forgetting compared to static baselines, with improvements in overall generalization scores. Empirical drift analysis shows that parameter importance overlap between early and late training stages is low (specific overlap ratios are reported), confirming that static masks become stale. EPI achieves the best or near-best performance across evaluated tasks, with aggregate performance gains over static isolation methods reported in the range of several percentage points on key benchmarks. (Note: precise per-benchmark numbers were not disclosed in the abstract; figures are derived from the paper's experimental findings.)
🏁 结论 EPI证明了将参数重要性视为动态演化量（而非固定属性）能够在多任务微调中持续取得更优结果，有效降低任务干扰与灾难性遗忘。本文确立了"隔离机制须与参数重要性的时序漂移同步"这一原则性且有效的大语言模型微调策略。 EPI proves that treating parameter importance as a dynamic, evolving quantity — rather than a fixed property — leads to consistently better multi-task fine-tuning outcomes, reducing both interference and forgetting. The work establishes that synchronizing isolation mechanisms with the temporal drift of parameter importance is a principled and effective strategy for large language model fine-tuning.

⚙️ Method · 方法详解

EPI提出一种动态微调框架，在整个训练过程中持续监控参数重要性。每隔固定周期，框架会基于梯度信号计算每个参数的重要性得分，并动态更新二值隔离掩码：重要性高的参数被保护，免受任务干扰；重要性下降的参数则被释放回可训练池。这种周期性的掩码刷新机制让模型能够追踪任务关键参数的动态变化。梯度信号作为轻量级的在线重要性代理，无需从头昂贵地重新计算。 EPI introduces a dynamic fine-tuning framework that continuously monitors parameter importance throughout training. At regular intervals, it computes gradient-based importance scores for each parameter, then updates binary isolation masks: high-importance parameters are protected from task-interfering updates, while parameters that have dropped in importance are released back into the trainable pool. This periodic mask refresh allows the model to track the evolving landscape of what matters for each task. The gradient signal serves as an online, lightweight proxy for importance without requiring expensive recomputation from scratch.

📐 Key Formulas · 关键公式

\[M_t = \mathbf{1}\left[s_t(\theta) \geq \tau\right], \quad s_t(\theta) = \left|\frac{\partial \mathcal{L}}{\partial \theta}\right|\]
💡 EPI的隔离掩码M_t由参数梯度绝对值（即重要性得分s_t）与阈值τ比较后动态生成，每隔周期T更新一次，重要性高的参数被隔离保护。 EPI's isolation mask M_t is dynamically generated by comparing each parameter's gradient-based importance score s_t (gradient magnitude) against a threshold τ; the mask is refreshed periodically, protecting high-importance parameters from interfering updates.

🌟 Why It Matters · 为什么重要

随着大语言模型越来越多地在多任务、多数据集上混合微调，管理任务干扰与灾难性遗忘成为部署中最迫切的实际挑战之一。EPI提供了一种轻量、有原则的解决方案，可无缝叠加到现有训练流程中而无需改变模型架构，对现实中的多任务LLM开发具有极高的实用价值。 As LLMs are increasingly fine-tuned on mixtures of tasks and datasets, managing interference and forgetting is one of the most pressing practical challenges in deployment. EPI offers a lightweight, principled solution that can be layered onto existing training pipelines without architectural changes, making it highly applicable to real-world multi-task LLM development.

MAny: Merge Anything for Multimodal Continual Instruction Tuning MAny：面向多模态持续指令微调的任意知识合并框架

🧑‍🔬 Zijian Gao, Wangwang Jia, Xingxing Zhang, Pengfei Qian et al.

图 Figure · Page 2 of paper

🎯 免训练模型合并，解决多模态持续微调中的双重遗忘问题
Training-free model merging that eliminates dual forgetting in multimodal continual instruction tuning.
维度 中文 English
❓ 待解决
的问题		 多模态大模型（如视觉语言模型）在依次学习新任务时，会严重"遗忘"之前学到的知识，即灾难性遗忘问题。现有方法主要只关注语言推理部分的遗忘，却忽略了另一个关键来源：视觉与语言之间的跨模态投影层也会发生"感知漂移"。这种"双重遗忘"——感知层面的漂移加上推理层面的崩塌——会随着任务增加而显著拖累模型性能。 When Multimodal Large Language Models (MLLMs) are fine-tuned sequentially on new tasks, they suffer from catastrophic forgetting — losing previously learned knowledge. Existing solutions mostly focus only on the language reasoning backbone, ignoring a second, equally damaging source of forgetting: drift in the cross-modal projection layer that connects vision and language. This dual-forgetting problem — perceptual drift plus reasoning collapse — degrades model performance significantly over time.
💡 解决
的亮点		 • **双重遗忘的首次揭示**：首次同时识别并解决跨模态投影空间的感知漂移与低秩参数空间的推理崩塌这两类遗忘现象。 • **CPM（跨模态投影合并）**：利用视觉原型引导，自适应地合并跨模态视觉表征，免训练恢复感知对齐能力。 • **LPM（低秩参数合并）**：利用递归最小二乘法对LoRA类低秩模块进行合并，给出闭式最优解，消除任务间互相干扰。 • **完全免训练**：所有合并操作通过高效的CPU代数运算完成，初次任务微调之后无需任何梯度计算。 • **Dual-forgetting discovery**: First work to explicitly identify and address both perceptual drift in the Cross-modal Projection Space AND reasoning collapse in the Low-rank Parameter Space simultaneously. • **CPM (Cross-modal Projection Merging)**: Uses visual-prototype guidance to adaptively merge cross-modal visual representations, recovering perceptual alignment without re-training. • **LPM (Low-rank Parameter Merging)**: Applies recursive least squares to merge LoRA-style low-rank modules with a closed-form, mathematically optimal solution, eliminating inter-task interference. • **Fully training-free**: All merging is done via efficient CPU-based algebraic operations — no additional gradient computation needed beyond the initial per-task fine-tuning.
🧪 实验
内容		 实验在多模态持续指令微调专用的UCIT基准上进行，测试在两种不同的多模态大模型骨干上依次学习多个任务后的最终平均准确率。对比了当前最优的持续学习基线方法，并在多个多模态基准上进行了泛化性验证，综合评估MAny在不同模型和任务序列下的鲁棒性。 Experiments were conducted on the UCIT benchmark for Multimodal Continual Instruction Tuning, evaluating sequential task learning across multiple MLLMs (at least two different backbone models). The framework was compared against state-of-the-art continual learning baselines. Primary evaluation metric is final average accuracy after all tasks have been learned sequentially, with additional robustness tests across diverse multimodal benchmarks to validate generalization.
📊 实验
结果		 在UCIT基准上，MAny相比当前最优方法在第一种多模态大模型上最终平均准确率提升高达 **+8.57%**，在第二种多模态大模型上提升 **+2.85%**，在两种骨干架构上均刷新了多模态持续指令微调的最优记录。 On the UCIT benchmark, MAny achieves up to **+8.57%** improvement in final average accuracy over the best existing method on one MLLM backbone, and **+2.85%** on a second MLLM backbone. These margins represent the best reported results on UCIT, establishing a new state-of-the-art for multimodal continual instruction tuning across both backbone architectures tested.
🏁 结论 MAny证明了同时解决感知漂移与推理崩塌这两类遗忘问题，比仅处理其中一类的现有方法有显著优势。该框架在保持计算轻量的前提下树立了多模态持续指令微调的新标杆，为在持续演化的真实场景中部署多模态大模型提供了实用且有理论保障的解决方案。 MAny proves that addressing both perceptual drift and reasoning collapse together — through training-free cross-modal and low-rank parameter merging — substantially outperforms methods that treat only one forgetting source. The framework establishes a new state-of-the-art for MCIT while being computationally lightweight, making it a practical and principled solution for deploying MLLMs in continually evolving real-world environments.

⚙️ Method · 方法详解

MAny的核心思路是：为每个任务单独维护对应的参数模块，推理时再通过两个互补组件将它们合并。对于视觉-语言桥接层（跨模态投影），CPM从每个任务收集视觉原型，以此为引导对投影权重进行自适应加权合并，确保模型能正确"看到"所有任务的视觉特征。对于语言推理侧，LPM将多个任务的LoRA模块合并问题建模为矩阵回归，用递归最小二乘法给出闭式解，找到同时最优近似所有任务适配的低秩权重矩阵。由于两部分都是纯线性代数运算，整个合并流程在CPU上完成，无需反向传播，轻量且可扩展。 MAny works by maintaining separate task-specific modules and then merging them at inference time using two complementary components. For the visual-language bridge (cross-modal projection), CPM collects visual prototypes from each task and uses them to guide an adaptive weighted merging of projection weights, ensuring the model can still correctly "see" features from all tasks. For the language reasoning side, LPM treats the collection of task-specific LoRA modules as a matrix regression problem and solves it in closed form using recursive least squares, finding the single low-rank weight matrix that best approximates all task adaptations simultaneously. Since both operations are pure linear algebra, the entire merging pipeline runs on CPU without any backpropagation, making it lightweight and scalable.

📐 Key Formulas · 关键公式

\[\mathbf{W}^*_{\text{LPM}} = \arg\min_{\mathbf{W}} \sum_{t=1}^{T} \| \mathbf{W} \mathbf{X}_t - \mathbf{W}_t \mathbf{X}_t \|^2_F\]
💡 LPM的核心优化目标：找到一个统一的低秩权重矩阵W*，使其在所有任务的输入分布上与各任务专属权重矩阵的输出尽可能接近，用递归最小二乘法给出闭式解。 The LPM objective: find a single merged weight matrix W* that minimizes the total Frobenius-norm deviation from each task-specific weight matrix across all task input distributions, solved in closed form via recursive least squares.
\[\mathbf{W}^*_{\text{CPM}} = \sum_{t=1}^{T} \alpha_t \mathbf{W}^{\text{proj}}_t, \quad \alpha_t = \frac{\exp(\text{sim}(\mathbf{p}, \boldsymbol{\mu}_t))}{\sum_{j} \exp(\text{sim}(\mathbf{p}, \boldsymbol{\mu}_j))}\]
💡 CPM的自适应合并公式：根据当前输入视觉特征p与各任务视觉原型μ_t的相似度，计算softmax权重α_t，对各任务的跨模态投影权重进行加权融合。 CPM adaptive merging: task-specific projection weights are combined using softmax weights α_t computed from the cosine similarity between the current visual feature p and each task's stored visual prototype μ_t, ensuring the merged projection is guided by perceptual relevance.

随着多模态大模型被部署到需要持续学习新任务的真实场景中，灾难性遗忘是核心瓶颈，MAny提供了一个免训练、有数学保障的实用解决方案。通过揭示并解决此前被忽视的感知遗忘维度，这项工作重新定义了多模态系统持续学习的研究框架。 As MLLMs are increasingly deployed in real-world applications that require learning new tasks over time, catastrophic forgetting is a fundamental bottleneck — MAny provides a training-free, mathematically grounded solution that is immediately practical. By exposing and tackling the overlooked perceptual forgetting dimension, this work reshapes how the community should think about continual learning for multimodal systems.
